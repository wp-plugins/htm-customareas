Custom areas allows your editors and contributors to create custom areas posts which the admin can then output inside their own posts and sections which allows
more control over what is output on the site. 
== Description ==

Have you ever wanted to allow your site users to add content to your site without giving them full access to all of the posts? Well now you can with HTM Custom Areas for wordpress. 
This plugin allows the administrator to select which user types can create custom area posts, the administrator can then use the shortcode icon on the editor to insert the custom post
 content inside of their own posts. To add this to a sidebar the administrator can create a text widget and place the shortcode inside of this. 



== Installation ==

1. Upload the folder htm_customareas to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin options under the 'custom areas' menu item
4. Create custom areas then output them using the shortcode icon on the editor toolbar on a post.
